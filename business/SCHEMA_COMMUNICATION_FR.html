MwangazaMail

Schéma du Parcours de Signalement

Flux de communication entre le citoyen et la plateforme — Mai 2026
📱
Étape 1 — Citoyen
L'utilisateur envoie un message
Depuis son téléphone, le citoyen envoie un message décrivant un incident (corruption, abus, surfacturation…) au numéro WhatsApp officiel de MwangazaMail.
Message WhatsApp
📞
Étape 2 — Réception
Numéro WhatsApp MwangazaMail
Le message est reçu par le numéro officiel de la plateforme. Le système accuse réception et engage une conversation guidée pour recueillir les détails du signalement.
Traitement automatique
🤖
Étape 3 — Intelligence Artificielle
Analyse et capture du signalement
L'intelligence artificielle analyse le message, identifie la catégorie (corruption, abus d'autorité, détournement…), la ville, l'institution concernée et le niveau de gravité.
Envoi sécurisé vers le cloud
☁️
Étape 4 — Base de données cloud
Enregistrement du signalement
Le signalement structuré est enregistré de manière sécurisée dans la base de données cloud de MwangazaMail. Un numéro de référence unique est généré automatiquement.
Mise à jour en temps réel
🖥️
Étape 5 — Tableau de bord
Visible sur le site web administrateur
Le signalement apparaît instantanément dans le tableau de bord sécurisé, accessible aux institutions abonnées. Elles peuvent filtrer, réviser et suivre le statut de chaque incident.
Confirmation WhatsApp
✅
Étape 6 — Confirmation
Numéro de référence envoyé au citoyen
Le citoyen reçoit automatiquement sur WhatsApp son numéro de référence unique (ex. #2891). Il peut l'utiliser pour suivre l'évolution de son signalement.

Légende

Citoyen : Toute personne disposant d'un téléphone et de WhatsApp, sans inscription ni application supplémentaire.
WhatsApp : Canal de communication principal. Fonctionne sur réseau 2G/3G. Disponible en français, lingala, swahili.
Intelligence Artificielle : Traitement automatique du message pour structurer et classifier le signalement sans intervention humaine.
Cloud : Infrastructure sécurisée hébergée aux États-Unis. Haute disponibilité, données chiffrées.
Tableau de bord : Interface web réservée aux institutions abonnées. Accès sécurisé par identifiant et mot de passe.
